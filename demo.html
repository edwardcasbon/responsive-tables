Responsive tables.

All major browsers & IE9+ - Basically all browsers that support media queries

The responsive tables CSS used here utilises media queries to collapse tables into stackable table cells at a defined width.

When marked up correctly, the column headings will be inserted into the table cell when it collapses. This is done by displaying the cell's "title" attribute in a "before" pseudo-element.

The "before" pseudo-element acts as a separate element and therefore can be style appropriately to your requirements.

Using CSS only

This table uses CSS only to add the column headings to the cell. In order for this to work the "title" attribute must be added to each cell with the value being the column heading.

Sample table: Areas of the Nordic countries, in sq km
Country Total area Land area
Denmark 43,070 42,370
Finland 337,030 305,470
Iceland 103,000 100,250
Norway 324,220 307,860
Sweden 449,964 410,928

Using the JavaScript polyfill

This table uses the JavaScript polyfill to add the column headings to the cell, rather than having to manually add the "title" attribute to every data cell.

Note: Only include the JavaScript polyfill if you don't add the headings using the "title" attribute, otherwise you don't need it.
